
## 2025-03-20 - Adding Semantic Landmarks to Dashboard Instruments
**Learning:** Complex multi-screen interfaces (e.g., flight simulator panels like ND or EICAS) benefit significantly from semantic grouping using `role="group"` and descriptive `aria-label`s, rather than relying solely on visual layout.
**Action:** Always add semantic structural landmarks (`role="group"` with `aria-label`) to distinct dashboard regions or panels when adding or modifying complex UI overlays to improve screen reader navigation.
